# sistemaGestionEnvios
# Sistema de Gestión de Envíos

Este proyecto consiste en una aplicación de consola desarrollada en **C#** que aplica el paradigma de **Programación Orientada a Objetos (POO)**. El sistema resuelve la problemática de la gestión logística de paquetes, integrando el uso de clases, objetos y principios fundamentales del paradigma.

## Funcionalidades del Sistema

El sistema cumple con los requisitos mínimos de operación mediante las siguientes funciones:

**Registrar datos:** Permite ingresar la información de nuevos paquetes al sistema, clasificándolos como nacionales o internacionales según sus características.
**Mostrar información:** El programa presenta en pantalla una lista con todos los envíos que se encuentran registrados actualmente.
**Buscar registros:** Permite localizar un paquete específico de forma rápida ingresando únicamente su número de guía único.
**Modificar datos:** El sistema da la opción de editar la información de un envío ya registrado si es necesario realizar algún cambio.
**Eliminar registros:** Permite borrar permanentemente un paquete de la lista cuando ya no se necesite tener su registro en el sistema.

## Tecnologías y Conceptos Aplicados

Para el desarrollo de esta solución se implementaron los siguientes pilares técnicos:
**Programación Orientada a Objetos:** Uso de clases, objetos, herencia, polimorfismo y encapsulamiento.
**Manejo de Colecciones:** Uso de listas u otras estructuras dinámicas para la manipulación de datos.
**Persistencia de Datos:** Almacenamiento y carga de información mediante archivos externos (txt o json).
